# Валидация формы отправки обращения

## Обзор изменений

1. Убрана загрузка фото/видео
2. Заменена геолокация на поле ввода адреса
3. Добавлена валидация всех полей
4. Добавлены маски для телефона

## Валидация полей

### Адрес
- **Обязательное поле**
- Минимум 10 символов
- Пример: "г. Чебоксары, ул. Ленина, д. 1"

### Описание проблемы
- **Обязательное поле**
- Минимум 20 символов
- Подробное описание проблемы

### ФИО (если не анонимно)
- **Обязательное поле**
- Минимум 3 символа
- Только буквы (кириллица и латиница), пробелы и дефисы
- Пример: "Иванов Иван Иванович"

### Телефон (если не анонимно)
- **Обязательное поле**
- Формат: +7 (XXX) XXX-XX-XX
- Автоматическая маска при вводе
- Валидация: 11 цифр, начинается с 7

### Email (необязательное)
- Стандартная валидация email
- Формат: user@example.com

## Маска телефона

Телефон автоматически форматируется при вводе:
- Ввод: `79001234567`
- Отображение: `+7 (900) 123-45-67`

## Отображение ошибок

Ошибки отображаются:
- При потере фокуса (onBlur)
- При попытке отправки формы
- Красная рамка вокруг поля
- Иконка AlertCircle с текстом ошибки

## Сохранение адреса

Адрес сохраняется в поле `address` таблицы `complaint_details` и добавляется в описание обращения для отображения в CRM.

## Координаты

Для демо используются фиксированные координаты центра Чебоксар:
- Широта: 56.1326
- Долгота: 47.2357

В production можно добавить геокодирование адреса через API (Yandex Maps, Google Maps, etc.)

## Пример использования

```typescript
// Валидация телефона
const validatePhone = (phone: string): boolean => {
  const cleaned = phone.replace(/\D/g, '')
  return cleaned.length === 11 && cleaned.startsWith('7')
}

// Маска телефона
const formatPhone = (value: string): string => {
  const cleaned = value.replace(/\D/g, '')
  if (cleaned.length === 0) return ''
  if (cleaned.length <= 1) return `+${cleaned}`
  if (cleaned.length <= 4) return `+${cleaned.slice(0, 1)} (${cleaned.slice(1)}`
  // ... и так далее
}
```
